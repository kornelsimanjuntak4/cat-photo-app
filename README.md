# Cat Photo App - Responsive Web Design Project

Aplikasi web ini adalah proyek praktik yang saya selesaikan sebagai bagian dari kurikulum **Responsive Web Design** di [freeCodeCamp.org](https://www.freecodecamp.org/). Melalui kursus ini, saya dibimbing untuk memahami fondasi utama dalam membangun halaman web yang modern dan aksesibel.

## 🎓 Hasil Pembelajaran
Proyek ini merupakan hasil bimbingan intensif yang mencakup:
- **HTML5 Semantik:** Mempelajari cara menggunakan elemen seperti `<main>`, `<section>`, dan `<figure>` untuk struktur kode yang lebih rapi dan terbaca oleh mesin.
- **Manajemen Form:** Membangun formulir interaktif dengan berbagai tipe input seperti *radio buttons*, *checkboxes*, dan *text input*.
- **Aksesibilitas Web:** Menerapkan atribut `alt` pada gambar dan struktur heading yang benar untuk mendukung perangkat pembaca layar (*screen readers*).
- **Struktur Dokumen:** Mengatur metadata dokumen pada bagian `<head>` termasuk pengaturan `charset="utf-8"`.

## 🛠️ Teknologi yang Digunakan
- **HTML5:** Fokus utama pada penataan struktur konten dan elemen formulir.
